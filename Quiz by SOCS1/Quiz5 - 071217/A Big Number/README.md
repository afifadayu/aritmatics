# A Big Number

## Given​ ​a​ ​number​ ​N.​ ​Your​ ​task​ ​is​ ​to​ ​compute​ ​F(N),​ ​where:

> If​ ​N​ ​>​ ​0,​ ​F(N)​ ​=​ ​F(N/3)​ ​+​ ​F(N/4)​ ​+​ ​1  
> Else​ ​F(N)​ ​=​ ​1  

### Input:

> The​ ​input​ ​consists​ ​of​ ​1​ ​integer​ ​N​ ​(0 ≤ N ≤ 2^30)​ ​that​ ​represents​ the ​numbers N.  

```sh
*SAMPLE INPUT*

1st Sample:
1

2nd Sample:
10

3rd Sample:
100
```

### Output:

> Output​ ​F(N)  

```sh
*SAMPLE OUTPUT*

1st Sample:
3

2nd Sample:
9

3rd Sample:
33
```
